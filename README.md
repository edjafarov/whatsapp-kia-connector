# whatsapp-kia-connector

An unofficial, self-contained WhatsApp connector for [alpha-cent / KIAgent](https://github.com/edjafarov/alpha-cent). It links a companion device via the [Baileys](https://github.com/WhiskeySockets/Baileys) library and ingests your chats (and an offline chat-export importer) into your local KIAgent corpus.
